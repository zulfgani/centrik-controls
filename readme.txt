Easily control Featherlite's Action hooked content by turning them on/off and relocating them.

== Description ==

Featherlite Controls brings the power of element location manipulation right inside the site customizer to allow ease move, enable/disable elements

== Installation ==

1. Install via the ClassicPress plugin installer or by uploading the files to your server.
2. Activate the plugin through the 'Plugins' menu in ClassicPress
3. Visit the site customizer and let the magic happen!

== Frequently Asked Questions ==

= Do I need Featherlite? =
Yes, in order to use this plugin your website will need to be running on the FeatherLite theme or a child theme.

= Where do I find the controls? =
Controls are located their respective Panels in the site Customizer which can be reached via the Appearance >> Customizer link in the dashboard.

= I've turned off an element/location but now my child theme additions have also disappeared, why is that? =
If you've manipulated a location with custom content via a child theme and/or addon plugin the it's best not to disable the element. Use the remove_action() to remove unwanted content instead.
